# IRBIS — Direção Visual para Criativos

## DNA Visual

| Elemento | Valor |
|---|---|
| Background | #0C0C0E — quase preto, não preto total |
| Acento | #FF3D00 — laranja-vermelho, usado com parcimônia |
| Texto | #FAFAFA — branco suave |
| Secundário | #8A8A93 — cinza médio para subinformação |
| Tipografia | Sora 800, maiúsculas, tracking apertado (-.04em) |
| Estilo | Dark studio — premium sem ostentação |
| Sensação | Confiança técnica. Sem frescura. Sem excesso. |

## Direção Estética

**O que é:** estúdio de design de alto nível que não precisa gritar para ser levado a sério. Cada pixel tem intenção.

**Referências visuais:** Apple keynote slides × editorial de arquitetura × interface SaaS premium × editorial de moda masculina escandinavista.

**O que evitar:** gradientes coloridos, sombras excessivas, ícones decorativos, textura de papel, fundos brancos, animações desnecessárias.
